# Stata Software — Econometrics & Data Analysis Environment

## About Stata Software

Stata Software provides a complete econometric analysis environment for linear/nonlinear regression, panel data, time series, and instrumental variables. Stata Software combines fixed/random effects, Arellano‑Bond, and VAR models. Stata Software supports simultaneous equations (IV, 2SLS, 3SLS), GMM, and cointegration directly into the interface. Stata Software integrates propensity scoring, marginal effects, robust standard errors, and error correction models. Stata Software enables users to analyze economic data, estimate causal effects, and generate publication‑ready output. Stata Software runs on portable workspaces and dedicated research stations. Stata Software is trusted by economists, financial analysts, policy researchers, and social scientists worldwide.

## Deep Dive — Econometric Methods Notes

Linear regression: OLS (ordinary least squares), weighted least squares, robust standard errors (Huber‑White, cluster‑robust), heteroskedasticity tests (Breusch‑Pagan, White)
Nonlinear regression: probit, logit (binary/multinomial/ordered), tobit, truncated regression, heckman selection, count models (Poisson, negative binomial)
Panel data: fixed effects (within estimator), random effects (FGLS), Hausman test, between effects, pooled OLS, dynamic panel (Arellano‑Bond, Blundell‑Bond, system GMM)
Time series: ARIMA, VAR (vector autoregression), VECM (vector error correction), cointegration tests (Engle‑Granger, Johansen), unit root tests (ADF, Phillips‑Perron, KPSS), impulse response functions, forecast error variance decomposition
Instrumental variables: 2SLS (two‑stage least squares), 3SLS (three‑stage least squares), GMM (generalized method of moments), weak instrument diagnostics (F‑statistic, Cragg‑Donald)
Post‑estimation: marginal effects (average marginal effects, marginal effects at means), predictive margins, contrasts, and hypothesis testing (Wald, likelihood ratio, F‑test)
